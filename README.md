# Homebrew Tap for Lattice

Official [Homebrew](https://brew.sh) tap for the [Lattice](https://lattice-lang.org) programming language.

## Install

```bash
brew tap alexjokela/lattice
brew install lattice
```

## Upgrade

```bash
brew update
brew upgrade lattice
```

## Usage

```bash
clat --version          # Print version
clat                    # Launch the REPL
clat program.lat        # Run a Lattice program
```
